# query

SQL-like query executor

## Query language

It's *like* SQL, but not SQL. See below.

## Supported features

* `SELECT *` without a GROUP BY.
* `LIMIT*`

## Unsupported features

These are unsupported *at the moment*.

* `GROUP BY`
* `JOIN`
* `ORDER BY`
